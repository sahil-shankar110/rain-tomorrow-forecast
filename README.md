**83.5% accurate ML weather forecasting app.** Live OpenWeatherMap API → RandomForest prediction → Rain tomorrow probability.

## 🎯 Features
- 🌍 **Live Weather Data** for 100+ cities worldwide
- 🤖 **RandomForestClassifier** trained on real weather dataset
- 📊 **Real-time Metrics**: Temperature, humidity, wind + confidence score
- ⚡ **Instant Deployment** on Streamlit Cloud
- 📱 **Responsive Design** for desktop/mobile

## 🚀 Live Demo

## 🏗️ Tech Stack
**Frontend: Streamlit**
**ML: scikit-learn (RandomForestClassifier)**
**Data: pandas, numpy**
**API: OpenWeatherMap + requests**
**Deployment: Streamlit Cloud + GitHub Actions**
